Pergunta:

O que é uma regra CSS? Explique o conceito de regra CSS e forneça um exemplo prático que inclua um seletor, uma propriedade e um valor.
Resposta:

Uma regra CSS é uma regra de estilo para a página WEB. Essa regra é feita de seletor (elemento que vai ser estilizado), propriedade (característica que vai mudar) e valor(como essa propriedade vai ser).

Exemplo:

p{ color: blue; }
