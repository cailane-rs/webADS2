CSS INLINE: Nesta forma, o CSS é aplicado diretamente na tag (elemento) HTML. Este formato não é indicado para ser utilizado, pois pode aumentar significativamente o tamanho do arquivo HTML, dificultando a manutenção futura necessária no arquivo HTML ou mesmo no código CSS.

CSS INTERNAL: Neste modelo, o CSS é aplicado dentro do "head", deixando o código mais organizado e também mais rápido em comparação ao CSS inline. No entanto, ainda não é o mais indicado.

CSS EXTERNAL: neste modelo, o código CSS é separado em um arquivo com a extensão .css. Este método é muito mais performático do que os demais listados acima. Para manutenção, é muito melhor, já que apenas o arquivo CSS carrega os dados.
